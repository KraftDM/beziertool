# Beziertool
Дипломный проект (Бакалавр 2020) - **Создание Easing анимаций**  
ФИО - Крафт Дмитрий Викторович

## Функционал
* Создание кривых безье для анимации
* Экспорт анимации в виде css кода
* Сохранение анимаций в архиве
